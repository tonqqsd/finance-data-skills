# Alpha Vantage API Reference

This reference documents the common functions available via the Alpha Vantage API.

## Core Financial Functions

- **GLOBAL_QUOTE**: Latest price and volume information.
- **TIME_SERIES_INTRADAY**: Intraday (1min, 5min, etc.) prices.
- **TIME_SERIES_DAILY**: Daily historical prices (OHLCV).
- **SYMBOL_SEARCH**: Find ticker symbols by company name.
- **MARKET_STATUS**: Check if major global venues are open.

## Fundamental Data

- **COMPANY_OVERVIEW**: Company profile, PE ratio, Dividend Yield, etc.
- **INCOME_STATEMENT**: Annual/Quarterly income statements.
- **BALANCE_SHEET**: Annual/Quarterly balance sheets.
- **CASH_FLOW**: Annual/Quarterly cash flow statements.
- **EARNINGS**: Historical/Future earnings (EPS).
- **EARNINGS_CALENDAR**: Upcoming earnings reports.

## Economic Indicators

- **REAL_GDP**: Annual and quarterly Real GDP of the USA.
- **FEDERAL_FUNDS_RATE**: Daily/Weekly/Monthly federal funds rate.
- **CPI**: Consumer Price Index (inflation barometer).
- **UNEMPLOYMENT**: Monthly unemployment rate data.
- **TREASURY_YIELD**: US treasury yields (various maturities).

## Commodities

- **WTI**, **BRENT**: Crude oil prices.
- **NATURAL_GAS**: Henry Hub natural gas prices.
- **GOLD_SILVER_SPOT**: Live spot prices for gold and silver.

## Technical Indicators

- **SMA**, **EMA**, **WMA**: Moving averages.
- **RSI**: Relative Strength Index.
- **MACD**: Moving Average Convergence / Divergence.
- **ADX**: Average Directional Movement Index.
- **BBANDS**: Bollinger Bands.

For full parameter documentation, visit the [official Alpha Vantage documentation](https://www.alphavantage.co/documentation/).
